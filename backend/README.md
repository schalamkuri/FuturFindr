# FuturFindr Backend
### Launching the backend server
Run:<br />
`docker build -t futurfindr-backend -f Dockerfile .`<br /><br />
`docker run --rm -it -p 5000:5000 futurfindr-backend`<br /><br />

Go to [localhost:5000](localhost:5000) on your web browser.<br />
